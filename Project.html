Project Report

NeuroLight Reflex Trainer – Full Documentation

This project, developed for the CSE1910 (Project A) Final Credit, is a hands-on demonstration of programming, design thinking, and problem-solving. The NeuroLight Reflex Trainer is an interactive system that stimulates cognitive activity using light and motion.

Key components include:

  • ESP32-C3 microcontrollers with wireless ESP-NOW communication
  • LED visual cues on modular dummies
  • Ultrasonic sensors for real-time hand detection

The project demonstrates skills from Structured Programming 1, Client-Side Scripting, and Design Processes. It includes planning, modular development, interface building, and a final evaluation—all meeting required outcomes.
